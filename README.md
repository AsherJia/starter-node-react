# starter-node-react
a starter project structure for react-app

## Dependencies

please use the latest version of following tools:

1. React
1. Webpack
1. Babel
1. Koa
1. ES6

## Running

### Install dependencies

    npm install
    
### Build

    npm run build

### Start

    npm start
    
### Browse

Open your browser, and visit `http://localhost:3000` to check the result
